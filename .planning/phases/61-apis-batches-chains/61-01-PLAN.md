## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-61-01 | Denial of Service | `Batch.insert_stream/2` options | mitigate | This plan adds `inserted_count`, `insert_chunk_count`, `insert_failed_chunk`, `insert_failure`, and `insert_failed_at` so malformed chunk/count handling can fail visibly instead of leaving lock-pressure or hanging-state ambiguity. |
| T-61-02 | Elevation of Privilege | Chain dynamic builders | transfer | This plan does not execute chain builders; Plans 61-03 and 61-05 reject anonymous functions and require durable builder references. |
| T-61-03 | Information Disclosure | Upstream output handoff | transfer | This plan stores no upstream payloads; Plan 61-05 keeps output handoff behind `JobRecord` references. |
| T-61-04 | Repudiation | Missing/expired/oversized outputs | transfer | This plan creates no output lookup path; Plan 61-05 maps unavailable outputs to explicit recoverable errors. |
| T-61-SC | Tampering | Package installs | mitigate | No package-manager install tasks exist in this plan; RESEARCH.md Package Legitimacy Audit lists no packages. |
</threat_model>
