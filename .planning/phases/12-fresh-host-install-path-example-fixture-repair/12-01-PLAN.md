## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Installer -> fresh host source tree | Library code writes directly into a host application's config, router, and seam modules. |
| Generated seam modules -> operator authorization/redaction | Starter modules can accidentally over-grant access or misrepresent production safety if they are too magic. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-01 | Tampering | installer config insertion | mitigate | Exercise the real fresh-host path in ExUnit and assert exact generated config/router values so broken Igniter edits fail immediately. |
| T-12-02 | Elevation of Privilege | generated auth/display seam modules | mitigate | Generate thin TODO-style host-owned modules with explicit placeholders instead of permissive demo defaults, per D-05 and D-06. |
| T-12-03 | Denial of Service | fresh-host install command | mitigate | Reproduce the prior `configure_group/6` crash in the regression harness and require a green install run before phase completion. |
</threat_model>
