## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Host signal ingress -> runtime matching | Correlation data from the host is advisory until the runtime resolves durable workflow authority. |
| Canonical signal row -> duplicate or replay attempts | Evidence rows must not rewrite canonical truth or hide the replay story. |
| Advisory wakeups -> DB-first reconcile | PubSub or coordinator delivery is untrusted and must not be required for correctness. |
