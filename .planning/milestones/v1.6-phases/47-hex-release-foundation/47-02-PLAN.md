## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-47-04 | Tampering / Elevation | igniter dependency scope | mitigate | `only: [:dev, :test], runtime: false` prevents code-generation machinery (igniter) from entering adopter production releases (Pitfall 14). Verified by grep + hex.build dep qualifier. |
| T-47-05 | Spoofing | ExDoc source links | mitigate | `source_ref: "v#{@version}"` pins doc source links to the exact published tag, preventing drift to `main` (which could show unpublished code). |
| T-47-06 | Repudiation | SPDX license declaration | mitigate | `licenses: ["Apache-2.0"]` must match the verbatim LICENSE file from plan 01 (single source of legal truth). |
</threat_model>
