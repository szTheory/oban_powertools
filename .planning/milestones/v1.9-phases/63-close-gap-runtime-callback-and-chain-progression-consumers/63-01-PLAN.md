<objective>
Implement the `ObanPowertools.Plugin.CallbackDispatcher` Oban plugin to close the runtime gap for batch and chain callbacks.

Purpose: Callbacks for batch completion and chain progression are currently inserted but never processed automatically. This plugin will continuously poll `oban_powertools_callbacks` and safely delegate to the domain processors, surviving malformed events.
Output: A resilient, production-ready `Oban.Plugin` module and its integration test suite.
</objective>
